# file-xor-encryptor

This is a console-based C++ program that can either encrypt or decrypt files provided by the user using the XOR encyption model.

## Features

- Select between encrypting and decrypting files
- Renames encrypted / decrypted files to "decrypted_(filename)" or "encrypted_(filename)" respectively
- Custom encryption key specific to this program only

## How to Run Simulation
1. Clone the repository: https://github.com/Gabriel-Ajul/file-xor-encryptor.git

2. Compile: g++ xor_encryptor.cpp -o xor_encryptor

3. Run: ./xor_encryptor

## Techniques Employed:

- OOP
- C++ Classes and Objects
- Ifstream and ofstream objects
- Binary mode in if/ofstream
- XOR Encryption via sstream
- Basic I/O and conditionals
- GitHub version control
